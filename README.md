# Restaurant inventory app
Python / Django app to allow a restaurant to keep track of its inventory.

<p>A user will log in as either 'front of house' or admin.</p>
<p>A front of house user can:<p>
<ul>
  <li>View the menu</li>
  <li>View customer orders</li>
  <li>Create and modify customer orders</li>
</ul>
<p>Admin can:<p>
<ul>
  <li>View the menu</li>
  <li>Create, modify and remove dishes from the menu</li>
  <li>Modify the recipe of a dish</li>
  <li>View what ingredients are in stock</li>
  <li>Create, modify and remove ingredients from stock</li>
  <li>Buy ingredients based on a shopping list</li>
  <li>Directly edit stock quantity</li>
  <li>View profit & loss</li>
  <li>View most popular dishes</li>
 </ul>
  
<p>The app is working in its current form but page stying has not yet been applied.</p>
